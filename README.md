## 🌏 Overview

Budakattu-Sante is a modern offline-first Android marketplace designed to connect tribal cooperatives directly with urban buyers while eliminating middlemen exploitation.

The platform enables:
- Tribal leaders to manage seasonal forest produce
- Buyers to preorder authentic tribal products
- Fair-trade transparency using MSP indicators
- AI-powered accessibility using Gemini
- Offline-first operations for low-network regions

# 🏗️ Architecture

```text
UI → ViewModel → UseCase → Repository → Local DB / Firebase
```

The application follows:
- Clean Architecture
- MVVM Pattern
- Modularized Codebase
- Offline-first Repository System

---

# ⚡ Tech Stack

| Category | Technology |
|---|---|
| Language | Kotlin |
| UI | Jetpack Compose, Material 3 |
| Backend | Firebase Firestore |
| Authentication | Firebase Auth |
| Local Storage | Room Database |
| Background Tasks | WorkManager |
| Dependency Injection | Hilt |
| AI | Gemini API |
| Image Loading | Coil / Glide |
| State Management | Kotlin Flow & Coroutines |
| Version Control | Git & GitHub |
